• Installation step of project
1. First download xampp software and the install it.
2. Then paste the project folder on xapmm/htdocs folder.
3. Then open xampp control panel and start the module of apache and mysql
4. Then go to any type of browser and use this path to run a project
http://localhost/online-food-order/index.php
• How to add database in project
1. Go to the http://localhost/dashboard/
2. Then click on myphpadmin
3. Then click on new and create the database name .the database named as 
foodorder.
4. Then click on import . and import the file foodorder.sql and click on go
5. Database add to the project
6. Finish
